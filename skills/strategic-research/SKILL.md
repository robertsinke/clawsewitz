# Strategic Research

Run the `/cw-research` workflow. Read the prompt at `../../prompts/research.md` (or `../../commands/cw-research.md` in Claude Code) for the full procedure.

**Agents used:** `researcher` (parallel web search, source-fetching, local file reading).

**Output:** research brief with bottom line, key facts with citations, comparative table, open questions, and sources.
