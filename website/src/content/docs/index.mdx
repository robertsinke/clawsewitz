  </TabItem>

## The Strategy Loop

A self-contained **7-step chain** with MECE rigor, 70 frameworks, and one Clausewitzian structural lens per stage. Every engagement follows the same disciplined path.

<CardGrid>
  <Card title="1. Define" icon="pencil">
    Frame the problem using SCQ, HTDQ, or Outcome. Push back if the stated question doesn't serve the real goal.

    **Clausewitz:** *Strategy serves policy.*
  </Card>
  <Card title="2. Split" icon="puzzle">
    MECE decomposition via Driver Tree, Bucketing, or Hypothesis Tree. Auto-validated by `cw-mece-check`.

    **Clausewitz:** *Schwerpunkt* — name the center of gravity.
  </Card>
  <Card title="3. Analyse" icon="magnifier">
    Run analyses on each leg. 70+ frameworks: financial models, market sizing, competitive, 5 Whys, value chain.

    **Clausewitz:** *Moral forces* — don't miss what spreadsheets can't see.
  </Card>
  <Card title="4. Insight" icon="star">
    Mount so-whats on page archetypes: One-Pager, Canvas, 2x2 Matrix, Journey Map, Capability Map.

    **Clausewitz:** *Fog acknowledgement* — name what you don't know.
  </Card>
  <Card title="5. Story" icon="document">
    Partner-grade narrative via Minto Pyramid, SCQA, Story Spine, 5 Act, or StoryBrand.

    **Clausewitz:** *Boldness* — if nobody could disagree, it's not a recommendation.
  </Card>
  <Card title="6. Decide" icon="approve-check">
    Recommendation using Eisenhower, ABCD, Bezos, SPADE, or Evaluation Tree. Red-team adversarial review.

    **Clausewitz:** *Ends-Ways-Means* — if any is vague, the recommendation isn't ready.
  </Card>
  <Card title="7. Act" icon="rocket">
    Implementation plan: workstreams, milestones, first-action owner. Premortem failure narrative.

    **Clausewitz:** *Friction register + Reserves* — plans at 100% break on first contact.
  </Card>
</CardGrid>

## Quick start

```
claude
/clawsewitz Acme Design — subscriber decline, 5%/mo for 3 months, need turnaround
```

The agent creates a case workspace under `~/.claude/clawsewitz/cases/`, runs the 7-step chain in partner voice, and produces deliverables at each stage. Hooks enforce chain order, auto-commit to git, and check MECE rigor automatically.

| Command | What it does |
|---------|-------------|
| `/clawsewitz <brief>` | Start or continue a strategic engagement |
| `/cw-case list` | List, resume, or archive case workspaces |
| `/cw-frameworks browse` | Browse the 70-framework catalog |
