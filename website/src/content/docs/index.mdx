## What it does

A self-contained **7-step Strategy Loop** with MECE rigor, 70 frameworks, and one Clausewitzian lens per stage.

<CardGrid>
  <Card title="Define" icon="pencil">
    Frame the problem using SCQ, HTDQ, or Outcome. Clausewitz lens: *strategy serves policy*.
  </Card>
  <Card title="Split" icon="puzzle">
    MECE decomposition via Driver Tree, Bucketing, or Hypothesis Tree. Lens: *Schwerpunkt*.
  </Card>
  <Card title="Analyse" icon="magnifier">
    Run analyses with 70+ frameworks. Lens: *moral forces* — don't miss what spreadsheets can't see.
  </Card>
  <Card title="Insight" icon="star">
    Mount so-whats on page archetypes. Lens: *fog acknowledgement* — name what you don't know.
  </Card>
  <Card title="Story" icon="document">
    Partner-grade narrative via Minto Pyramid. Lens: *boldness* — hedge and you say nothing.
  </Card>
  <Card title="Decide" icon="approve-check">
    Recommendation using SPADE, Bezos, Evaluation Tree. Lens: *Ends-Ways-Means*.
  </Card>
  <Card title="Act" icon="rocket">
    Implementation plan. Lens: *friction + reserves* — plans at 100% break on first contact.
  </Card>
</CardGrid>
